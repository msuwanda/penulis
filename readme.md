# Mandala framework

framework php & reactJs 

## Getting Started


### Prerequisites

* [composer](https://getcomposer.org/download/)
* [web server](https://httpd.apache.org/)


### Installing

install package
```
composer install
```
setting database
```
php mandala database <host> <user> <password> <name>
```
install framework
```
php console install
```
